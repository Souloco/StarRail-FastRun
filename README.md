# 使用说明
- 游戏设置
    - 是否沿用自动战斗设置---是
    - 键鼠模式显示底部提示---是
    - 操作模式：疾跑---长按进入疾跑状态
    - 视角灵敏度---3
    - 游戏分辨率1920x1080---有窗口化选窗口化没有就选全屏幕
- 软件设置
    - 以管理员身份运行gui.exe
    - 英文路径下使用
